
sinatra microservice base
========================================

This project is a starting-point for any Ruby microservice which requires:

Application
-

  * sinatra
  * activerecord

Testing
-

  * factory_girl
  * rspec
  * database_cleaner

Development
-

  * Vagrant
  * knife-solo
  * chef-solo
  * librarian-chef
  * dotenv
  * shotgun
  * debugger
  * better_errors

Deployment
-

  * dotenv-deployment
  * puma
  * nginx
